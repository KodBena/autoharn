=== ROUND 2 (ledger row 1174, faithfulness re-check): missing/empty flag-value silently
=== substitutes the WRONG default and WRITES A REAL ROW for it ===

Legacy's own behavior on the identical input (recovered source, the shared top-of-file flag
loop's `--db-role) db_role="${2:-}"; shift 2 ;;`): the `${2:-}` assignment succeeds silently
(bash's own safe-default), but the immediately following `shift 2` FAILS when only the flag
itself remains unconsumed (shift count 2 > 1 remaining positional) -- under `set -euo pipefail`
this aborts the script: exit 1, nothing written, no teach-text. A wrong write beats a crash for
badness; this is the defect the crash was accidentally masking.
