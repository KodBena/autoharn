# autoharn (archive)

Development of autoharn has moved to branch **`main`**, starting at the `v1.0` tag — the
fresh-history public cut of this project, published 2026-07-15.

This branch (`master`) is retained as an **archive** of the project's pre-v1.0 development
history. It is not maintained going forward: no new commits, no issue triage, no support.
If you are looking for the working project, clone this repository and check out `main`
(or clone at tag `v1.0` or later) instead of `master`.

## Why the history restart

`main` begins from a single root commit rather than continuing this branch's history. The
development history that produced the `v1.0` snapshot remains in the maintainer's own
private archive; this `master` branch is that same pre-cut history, kept here, read-only,
for anyone who wants to see how the project got to `v1.0`.
