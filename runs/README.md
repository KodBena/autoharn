# runs/

NEW run and close records executed **from this repo** accrue here, one subdirectory per
run (`runs/<run-id>/`). This is the forward application of the mandate's
evidence-stays-where-it-happened rule ([C19]): after the HOME-FLIP, where a run happens
is autoharn. Old-repo run evidence never moves here — it stays in the read-only archives.
The first occupant is the mandate-§6 acceptance run.
